# Cardápio Digital com Supabase

## Instruções de Configuração

### 1. Criar Projeto no Supabase
1. Acesse [supabase.com](https://supabase.com) e crie uma conta gratuita
2. Crie um novo projeto
3. Acesse o **SQL Editor** e execute os comandos abaixo para criar as tabelas:

```sql
-- Tabela de Categorias
CREATE TABLE categories (
  id TEXT PRIMARY KEY,
  name TEXT NOT NULL,
  created_at TIMESTAMP DEFAULT NOW()
);

-- Tabela de Pratos
CREATE TABLE dishes (
  id BIGINT GENERATED BY DEFAULT AS IDENTITY PRIMARY KEY,
  name TEXT NOT NULL,
  price TEXT NOT NULL,
  description TEXT,
  category_id TEXT REFERENCES categories(id) ON DELETE CASCADE,
  image TEXT,
  hot BOOLEAN DEFAULT FALSE,
  popular BOOLEAN DEFAULT FALSE,
  created_at TIMESTAMP DEFAULT NOW()
);

-- Inserir categorias iniciais
INSERT INTO categories (id, name) VALUES
  ('entradas', 'Entradas'),
  ('porcoes', 'Porções'),
  ('principais', 'Pratos Principais'),
  ('frutosdomar', 'Frutos do Mar'),
  ('bebidas', 'Bebidas'),
  ('sobremesas', 'Sobremesas');
```

### 2. Configurar Políticas de Segurança (RLS)
No painel do Supabase, vá para **Authentication** > **Policies** e crie as políticas abaixo para cada tabela:

#### Para a tabela `categories`:
Crie uma política que permita **todas as operações** para **todos**:
- Policy name: `Permitir tudo`
- Operation: `ALL`
- Policy expression: `true`

#### Para a tabela `dishes`:
Crie uma política que permita **todas as operações** para **todos**:
- Policy name: `Permitir tudo`
- Operation: `ALL`
- Policy expression: `true`

### 3. Pegar Credenciais
No painel do Supabase, vá para **Project Settings** > **API** e copie:
- `Project URL` (ex: `https://xxxxxx.supabase.co`)
- `anon/public key` (a chave pública)

### 4. Usar o Site
1. Abra `index.html` no navegador
2. Insira as credenciais do Supabase no modal de configuração
3. Pronto! Agora todos os dados são salvos no Supabase

## Hospedagem
Você pode hospedar o site gratuitamente no:

Basta fazer upload dos 4 arquivos (`index.html`, `style.css`, `script.js`, `README.md`).
